# Project Workflow

## Objective

Compare dextral and sinistral Dia2 alleles to understand the molecular basis of shell coiling in *Lymnaea stagnalis*.

## Workflow

1. Retrieve nucleotide sequences from NCBI GenBank.
2. Retrieve corresponding protein sequences.
3. Compare dextral and sinistral alleles.
4. Identify the frameshift mutation.
5. Compare protein lengths.
6. Visualize conserved protein domains.
7. Interpret the biological consequence of domain loss.

## Software and Databases

- NCBI GenBank
- BLAST
- Clustal Omega
- NCBI Conserved Domain Database (CDD)
- Microsoft PowerPoint
